The Bezel Project is setup with the following sytem-to-core mapping.


To show a specific game bezel, Retroarch must have an override config file for each game.  These 
configuration files are saved in special directories that are named according to the Retroarch 
emulator core that system uses.


The supplied Retroarch configuration files for the bezel utility are setup to use certain 
emulators for certain systems.


In order for the supplied bezels to be shown, you must be using the proper Retroarch emulator 
for a system listed in the table below.


This table lists all of the systems that have the abilty to show bezels that The Bezel Project 
hopes to make bezels for.
